# ICL vs IOL 위치 차이

## ✔️ IOL (백내장 수술)

- 위치: **capsular bag (수정체낭 내부)**
- 기존 수정체 제거 후 → 그 자리에 삽입

```
각막 → 전방 → (빈 capsular bag) → IOL 삽입
```

## ✔️ ICL (Implantable Collamer Lens)

- 위치: **섬모체 고랑 (ciliary sulcus)**
- **자연 수정체는 그대로 유지**
- 그 앞쪽에 렌즈를 추가로 넣음

```
각막 → 전방 → 홍채 → ICL → 자연 수정체  
             ↑  
        ciliary sulcus에 고정
```
