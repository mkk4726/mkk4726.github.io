# 이력서 초안 (CJ대한통운 - 디지털운송플랫폼 Data Scientist)

## 포지션 핏 한 줄
- 실험 설계/통계 검증 역량과 운영형 ML 시스템 구축 경험을 결합해 플랫폼 의사결정을 고도화하는 포지셔닝.

## 상단 소개 문구 초안
데이터 기반 의사결정을 위해 지표 정의, 실험 설계, 예측 모델링, 운영 자동화를 통합적으로 수행해 온 데이터 사이언티스트입니다. 복잡한 현업 문제를 계량 가능한 과제로 전환하고, 통계적 검증과 시스템 구현을 연결해 실질적인 운영 개선 성과를 만들어왔습니다.

## 핵심 경력 포인트 (JD 매핑)
- 실험 설계 및 통계 검증
  - 문제 상황을 가설 형태로 분해하고, 핵심 지표를 재정의해 개선 효과를 검증.
  - 단순 전후 비교를 넘어 신뢰 가능한 평가 프레임을 설계해 의사결정 품질 향상.
- 예측/최적화 관점의 모델링 경험
  - 예측 불확실성을 고려한 접근으로 운영 리스크를 줄이는 모델링 수행.
  - 비즈니스 제약을 반영한 실용적 방법론 선택으로 실제 적용 가능성 확보.
- 데이터 파이프라인 및 운영 체계 구축
  - 데이터 수집-가공-학습-추론-모니터링 흐름을 End-to-End로 설계.
  - Airflow 중심 자동화와 품질 점검 체계로 지속 운영 가능한 구조 마련.
- SQL 기반 분석 및 데이터 구조화
  - 서비스 로그/업무 데이터를 분석 목적에 맞게 구조화하고 지표용 데이터셋 설계.
  - 분석 산출물이 실험/모델 개선으로 이어지는 루프를 구축.
- 협업 커뮤니케이션
  - 비기술 이해관계자와 기술 팀 사이에서 지표와 결과를 해석 가능한 언어로 전달.

## 기술 스택 강조
- Python, SQL, 통계적 가설 검증, 실험 설계
- 모델링/서빙/모니터링 파이프라인 설계
- 자동화 워크플로우 운영(Airflow 등)

## 자기소개서/면접용 포인트
- 수요/공급 예측, 운영 최적화 과제에 연결 가능한 문제 구조화 역량 강조
- 모델 성능뿐 아니라 운영 효율/리스크 관리 관점으로 성과 설명
- "분석"과 "구현"을 분리하지 않고 끝까지 책임졌던 사례 제시
