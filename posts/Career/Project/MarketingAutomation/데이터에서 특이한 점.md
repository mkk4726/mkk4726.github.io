## AI 분석 정리

**한 줄 요약:** 숫자가 튀어 보이지만, “비정상”이라기보다 **네이버 카페 규칙 + 시력교정 도메인**이 겹친 **구조적으로 나오기 쉬운 분포**에 가깝다. 다만 비율이 높아서 **해석·필터링**은 꼭 필요하다.

### 공통 패턴

세 카페 모두 **신규 가입·등업 성격 글**이 상위권이다.

| 카페 | 대표 카테고리 | 비고 |
|------|----------------|------|
| 라준사 | 가입인사 **21%** | |
| 라식구조대 | 자동등업도전! **29.6%** | |
| 강남안과정보공유 | 시력교정 등업 신청 **5.9%**, 노안백내장 등업 신청 **1.8%** | 정보 게시판 비중은 상대적으로 큼 |

**등업(게이팅):** 민감한 정보(비용, 병원 추천, 블랙리스트 등)를 **2·3단계 등급** 뒤에 두는 카페가 많다. 그래서 “인사 몇 번·댓글 몇 번” 같은 **규칙이 저관여 글을 대량 생산**한다. 흐름은 대략 **가입 → 등업용 글 → 이후 활동 감소/이탈**이 반복되면, 정보·상담 글보다 **등업·인사 글이 쌓이는 역전**이 생긴다.

### 카페별로 보면

- **라준사:** 가입인사 21% + 자유게시판 48.8% → **저관여·신규 유입 합쳐 ~70%** 부근.
- **라식구조대:** 자동등업도전 29.6% + 회원일상수다 16.8% 등 → **온보딩·가벼운 수다가 절반 이상**. `지인할인요청` **2.9%**는 거래·소개 성격 활동이 섞여 있다는 신호로 볼 수 있음.
- **강남안과정보공유:** 정보 공유·Q&A 비중이 상대적으로 크지만, 등업·수다와 **여전히 섞여 있음**.

### 왜 이런 구조가 나오나

1. **시스템이 만든 트래픽:** 등업 조건 때문에 생기는 글은 **오가닉 콘텐츠와 성격이 다름** (플랫폼·운영 규칙의 “행정적 노이즈”).
2. **일회성 라이프사이클:** 시력교정은 취미 커뮤니티와 달리 **정보 습득 → 결정 → 수술 → 이탈**이 많아 **이탈(천이율)이 높고**, 항상 **신규 유저 쪽이 두껍게** 보이기 쉽다.
3. **크롤링은 “빈도” 스냅샷:** 인사·등업 글은 쓰기 쉽고, 상세 후기는 시간이 든다. 같은 기간을 긁으면 **저비용 글이 고가치 글보다 개수로는 항상 우세**하기 쉽다.
4. **정보 비대칭:** 가격·병원·리스크 불확실 → 신규 질문·가입 활동이 많아진다.
5. **러커 구조:** 대부분은 안 쓰고, 글 쓰는 쪽은 **신규·이벤트·등업 목적**에 치우치기 쉽다.

### “특이하다”고 느낀 핵심

표면의 가입/등업 비율보다, 구조적으로는 **지속 사용자(고인물·장기 후기) 비중이 약한 편**일 가능성이 크다. 전형적인 장기 커뮤니티보다 **의사결정 전 정보 탐색·리드에 가까운 채널**에 가깝다.

**퍼널로 보면:** 상단(가입·인사·등업)이 두껍고, 하단(심화 후기)은 상대적으로 얇은 **Top-heavy funnel**.

### 데이터·모델링에서 주의할 점

- **활성도 = 글 수**로만 보면 **과대추정**될 수 있음.
- 텍스트/토픽 분석 시 가입인사·등업 글이 **주제 왜곡(노이즈)**을 일으킬 수 있음.
- **Signal / noise 분리:** 후기·상담·부작용 등은 signal, 가입인사·등업·(필요 시) 저관여 수다는 noise로 두고 **모델링 전 필터링**하는 편이 안전하다.
- 추천·검색 설계: **장기 사용자 모델**만으로는 부족하고, **콜드스타트·의도(intent) 분류**(예: 고민 단계 / 병원 비교 / 수술 후) 쪽이 더 맞을 수 있다.

가입·등업 카테고리를 빼고 보면, 지역·Q&A·후기 쪽 분포는 훨씬 “의료 주제에 가깝게” 정규화될 가능성이 있다.

### 결론

- **이상해 보이지만 비정상은 아님** — 등업 시스템, one-shot 사용자, 정보 비대칭이 겹친 결과.
- 해석상 핵심은 이 카페들이 **순수 커뮤니티 지표로 읽기 어렵고**, **의사결정 전 정보 탐색 채널**에 가깝다는 점.

**다음에 할 수 있는 것:** 가치 있는 글만 남기는 **필터링 규칙**, 추천/검색용 **데이터셋 설계**(의도 라벨, 등업글 제외 기준 등)를 목적에 맞게 구체화하기.
