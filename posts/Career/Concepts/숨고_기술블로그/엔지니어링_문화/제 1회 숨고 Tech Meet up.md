# 제 1회 숨고 Tech Meet up

## 원문

- 발행일: 2025.01.10
- 링크: https://soomgo.team/blog/posts/67281802de083ba1deb20ce1

## 한 줄 요약

숨고는 외부 개발자 대상 Front-end & Mobile Meet up을 열어, 내부 기술 개선 사례와 개발 문화를 공유했다.

## 행사 성격

- React / React Native 관심 개발자 약 30명 규모
- 첫 번째 Tech Meet up
- 발표 + 네트워킹 중심 구성

## 공유된 내용

### Front-end 세션

- 빌드 워크플로우 가속화
- 한 달 기준 약 `5,100분`의 시간과 월 `220만 원` 수준 비용이 배포 프로세스에 쓰이고 있음을 확인
- 스위칭 코스트까지 포함하면 비용이 더 커진다고 분석
- S3 업로드 속도 개선, GitHub Larger runner, Docker layer caching 등으로 개선

### Mobile 세션

- React Native 중심 기술 스택 소개
- 상태 관리, 빌드/배포, 모니터링 구성 공유
- App startup time 개선
- Small Release 대응을 위한 자동화와 롤백 프로세스
- Native Stack, Native Module 활용 사례

## 의미

- 숨고가 내부 기술 개선을 외부 커뮤니티와 공유하기 시작했다는 점에서 상징적이다.
- 단순 채용 이벤트보다 `기술 조직이 무엇을 중요하게 보는지`를 드러내는 자리로 읽힌다.
