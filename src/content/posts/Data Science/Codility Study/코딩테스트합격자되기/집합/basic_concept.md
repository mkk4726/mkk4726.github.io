# 개념 스케치

상호베타적 집합. 서로 겹치는게 하나도 없는 것.
-> 그래프에서 자주 사용됨. 사이클을 확인하는 일.

다음과 같은 부분에서도 사용됨.
- 이미지 분할
- 도로 네트워크구성
- 최소 신장 트리 알고리즘 구현
- 게임 개발
- 클러스터링 작업


배열로 트리를 표현 -> 집합을 표현.
대표원소.

파인드, 유니온.
랭크.
경로 합치기 (깊이가 깊어져서 탐색과정이 비효율적일 때 합쳐버리는 것)

유니온 -> 두 집단 중 하나의 대표원소를 다른 집단의 대표원소의 자식 노드로 바꾸는 것. 
랭크를 기준으로 더 높은 것에 낮은 것을 붙임.
